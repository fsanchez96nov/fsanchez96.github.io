la etiqueta video: para especificar la ubicacion del archivo que se reproducira, puede usar el elemento (source) dentro del elemento (video) (esto debe reemplazar el atributo scr en la etiqueta (video); de apertura.)Tambien puede utilizar varios elements (source) para especificar que el video disponible en diferentes formatos. (Debido a un error en el ipad,debe proporcionar el video MP4 como el primer formato. De lo contrario puede que no inicie)

src:este atributo especifica la ruta de acceso al video.

type: se debe utilizar este atributo para decirle al navegador que formato es el video. de lo contrario, descargara la parte del video para ver si se puede reproducir el archivo que tomara tiempo y ancho de banda.

codecs: el codec que se uso para codificar el video se suministra dentro del atributo type. tenga en cuenta que es uso de comillas simples, asi de comillas dobles en el atributo de tipo, cuando se suministra.
