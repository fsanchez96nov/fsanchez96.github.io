Letter And Word Spacing

by TPS Sena

El kerning es el termino que utililza los tipografos para el espacio de cada letra. Puede controlar el espacio entre cada letra con la propiedad: letter-spacing. es particularmente util para aumentar el kerning cuando su encabezado o frase esta en mayuscula. si el texto esta en un caso o normal, aumentar o disminuir el kerning puede dificultar la lectura.

El intervalo predeterminado

Cuando se espcifica un valor para esta propiedades, se debe dar en EMS. El intervalo predeterminado entre las palabras se establece por el tipo de letra (a menudo alrededor d e0.25 EM), y es poco probable que deba cambiar esta propiedad con regularidad.
