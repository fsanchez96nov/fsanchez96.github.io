Background color- Color de Fondo

Si no se especifica un Color de Fondo, el fondo es transparente.

CSS trata cada elemento HTML como un cuadro y la propiedad Background color establece el color del fondo para el cuadro. se puede especificar colores de primer plano: valores RGB, Codigo Hexadecimal y nombre de colores. tambien hemos utilizado la propiedad padding para separar el texto de los bordes de los cuadros. Esto hace que sea mas facil de leer.
