CSS3: hsl y hsla

la propiedad de color HSL se introdujo en CSS3 como una forma alternativa de especificar los colores. El valor de la propiedad comienza con las letras HSL, seguidas de valores individuales dentro de los parentesis para:

HUE
Esto se expresa como un angulo ( entre 0 y 360).

saturacion
Esto se expresa como un porcentaje.

ligereza
esto se expresa como un porcentaje con 0% siendo blanco, 50% siendo normal, y 100% siendo negro.

la propiedad de color HSLA permite especificar propiedades de color mediante matiz, saturacion y luminosidad como se indica anteriomente, y agrega un cuarto valor que representa la transparencia ( igual que la propiedad RGBA)
La a representa: alfa este de expresa como un numero entre 0 1.0 por ejemplo, 0.5 representa 50% de transparencia y 0.75 representa de transparencia
